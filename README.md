# Marketing
Marketing Campaign Performance Analysis (Dec 2025 - Feb 2026)
Project Overview
This project provides a comprehensive analysis of a 2-month marketing campaign. The goal was to evaluate budget efficiency, channel performance, and audience behavior to optimize future marketing spend and maximize ROI.
Key Insights (Executive Summary)
High Profitability: Achieved an 8x Return on Investment (ROI), justifying the budget overspend as a successful scaling strategy.

Market Dominance: Egypt emerged as the primary market, significantly outperforming other regions.

Channel Efficiency: Social Media (Meta & TikTok) drove 78% of total sales, proving much more cost-effective than Google Ads.

Optimization Opportunity: Identified a low CTR, indicating a need for refreshed ad creatives despite high reach.

Winning Audience: Lookalike audiences and the 18-24 age group showed the highest conversion rates.

Tech Stack
Analysis & Visualization: Power BI (Desktop & Service).

Data Modeling: Star Schema.

Languages: DAX (for complex measures like ROI, CPA, and Growth rates), SQL (Data Cleaning).

Version Control: Git (PBIP format).

Dashboard Features
Overview Page: High-level KPIs (Total Spend, Revenue, ROI, Total Conversions).

Channel Analysis: Comparison between Google Ads vs. Meta (CPA & ROAS).

Audience Insights: Demographic breakdown (Age, Gender) and Audience Type (Interest vs. Lookalike).

Geographic Map: Sales distribution across different regions.

Recommendations
Budget Reallocation: Shift a portion of the Google Ads budget to Instagram/TikTok to lower overall CPA.

Creative Refresh: Launch a new set of video/image assets targeting the 18-24 segment using a more relatable brand voice.

Scaling: Increase investment in Lookalike audience segments
